SecretVault Pages
  • Home page: JavaScript and CSS for the typing animation
  • About me: JQuery library for the UI element
  • Checklist: JQuery library for the UI element
  • Login: Implemented PHP for validating the login in MySQL DB and JavaScript for pop up alerts
  • Register: Membership registration form. PHP is used to create an account and save the users info in MySQL DB and JavaScript pops up an alert if the login gets created successfully
  • View all logins: PHP and MySQL
  • Add login: Form submission: PHP processes the form and stores the login in the DB
  • Add secret key: Form submission: PHP processes the form and stores the key in the DB
  • Add secret note: Form submission: PHP processes the form and stores the note in the DB
SecretVault Navigation
  • Login
    • If you have an existing account then you can login directly
    • If you dont have an account you need to register your account first and then you will be redirected to the login screen
  • Viewing the secrets
    • This is the default page that you see once you login
    • If this is your first time logging in then you will not see anything but if you are a returning user then you can see your stored secrets.
  • Add secrets
    • Click on add login to store a credential that contains a user and password
    • Secret key and secret note is similar howevever secret key is intended for storing product keys, activation key or something similar to that matter. Whereas secret key is used to store other secret notes such as social security number, secret questions for a specific logins, etc.
